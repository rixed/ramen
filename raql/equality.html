Equality test

Syntax

…expr… = …expr…

…expr… != …expr…

…expr… <> …expr…

Typing

t, t -> BOOL

Description

Test for equality.

Values of any type can be tested for equality, but both values must have the same type.

Examples

SELECT "foo" = "FOO"…

FALSE

SELECT [1; 2] <> [2; 1]…

TRUE

SELECT PI != 3.14…

TRUE
