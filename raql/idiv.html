Integer division

Syntax

…num-expr… // …num-expr…

Typing

num1, num2 -> largest(num1, num2)

Description

Divides two numeric values, truncating toward zero.

The result type is the largest of the operand types.

For FLOAT operands the result will still be a FLOAT, floored.

Examples

SELECT 10//3…

3

SELECT -10//3…

-3

SELECT 10.5//3.1…

3
