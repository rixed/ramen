Relative difference

Syntax

RELDIFF(…num-expr…, …num-expr…)

Typing

num1, num2 -> FLOAT

Description

Compare the two operands A and B by computing:

MIN(ABS(A-B), MAX(A, B)) / MAX(ABS(A-B), MAX(A, B))

Returns 0 when A = B.

Examples

SELECT RELDIFF(1, 1)…

0

SELECT RELDIFF(10, 9)…

0.1

SELECT RELDIFF(9, 10)…

0.1

SELECT RELDIFF(-9, -10)…

0.1

SELECT RELDIFF(1, -10)…

1.1
