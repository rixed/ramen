Comparison operators

Syntax

…expr… > …expr…

…expr… < …expr…

Typing

t, t -> BOOL

Description

Strict comparison operators.

Values of any type can be compared, but both values must have the same type.

Examples

SELECT 1 > 0…

TRUE

SELECT 1 > 1…

FALSE

SELECT TRUE > FALSE…

TRUE

SELECT "foo" < "bar"…

FALSE

SELECT (5; 1) < (5; 2)…

TRUE
