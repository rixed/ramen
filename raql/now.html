Return the current time as a UNIX timestamp

Syntax

NOW

Typing

float

Description

Floating point number of seconds since 1970-01-01 00:00 UTC.

Warning: Parents of a function using NOW can not be archived, as the NOW function is not deterministic. For better results, use NOW as early as possible in the processing stream.
