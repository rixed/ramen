Minimum

Syntax

MIN(…expr1…, …expr2…, …)

Typing

t1, t2, … -> largest(t1, t2, …)

Description

Return the minimum value of all the operands.

All operands must have compatible types (ie. it is possible to convert one into another).

The result will be NULL if any of the operands is.

Examples

SELECT MIN(1, 2, 3)…

1

SELECT MIN("foo", "bar")…

"bar"

SELECT MIN([5; 6], [3; 9])…

[3; 9]
